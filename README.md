# TikTok Tides
CSC316 Final Project (Fall 2025) | Team TrendTokers

## Overview
**TikTok Tides** is a project developed by **TrendTokers** to explore the lifecycle of TikTok trends and their impact on real-world behavior. As TikTok continues to shape popular culture, understanding the evolution of trends—how they emerge, spread, fade, and influence culture—is essential. This project collects and analyzes TikTok data to gain insights into how TikTok trends affect language, purchasing habits, fashion, and even product shortages.
